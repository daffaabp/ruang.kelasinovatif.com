tugas kamu adalah melakukan commit dan push ke github, lakukan langkah-langkah berikut ini:

1. sebelum melakukan commit, perintah "pnpm lint" untuk mengecek apakah ada error atau tidak dan jika ada error, kamu segera memberitahu user dan berhenti dan jangan lanjut ke step berikutnya
2. analisa semua perubahan yang ada yang diberikan oleh user
3. pikirkan cara mengelola commit sesuai dengan jenis perubahan, fitur, file grup, project struktur
4. git add, sesuai dengan pengelompokannya
5. git commit, sesuai dengan pengelompokannya, gunakan bahasa indonesia
6. tambahkan deskripsi yang message nya harus deskriptif, panjang, lengkap, detail, minimal 3 paragraf, dan gunakan bahasa indonesia
7. git push
8. pastikan semua langkah diatas dilakukan satu persatu jangan bersamaan
