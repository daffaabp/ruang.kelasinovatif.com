# KelasInovatif

Platform pembelajaran modern (v3.0) yang memberdayakan pendidikan melalui inovasi.

## 🚀 Fitur Utama

- **UI Modern**: Desain "Deep Green" dengan efek *glassmorphism* dan animasi latar belakang yang dinamis.
- **Landing Page Kompak**: Tampilan satu halaman (*single view*) yang responsif dan informatif.
- **Autentikasi Terlokalisasi**: Halaman Masuk dan Daftar yang sepenuhnya dalam Bahasa Indonesia.
- **Tech Stack Terkini**: Dibangun dengan Next.js 15, Tailwind CSS, dan Bun.

## 🛠️ Teknologi

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS (dengan `tailwindcss-animate`)
- **Runtime/Package Manager**: [Bun](https://bun.sh/)
- **Form Handling**: React Hook Form + Zod
- **Icons**: Material Icons & Lucide React
- **Notifications**: Sonner

## 📦 Instalasi & Menjalankan

Proyek ini menggunakan **Bun** sebagai package manager utama.

1. **Install dependensi**:
   ```bash
   bun install
   ```

2. **Jalankan server development**:
   ```bash
   bun dev
   ```

3. **Buka di browser**:
   Akses [http://localhost:3000](http://localhost:3000) untuk melihat aplikasi.

## 🎨 Tampilan

Halaman utama dan autentikasi telah didesain ulang untuk memberikan pengalaman pengguna yang premium dan fokus pada konten edukasi.
